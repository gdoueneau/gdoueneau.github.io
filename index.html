Research

See my list of publications.

My research deals with simple models of computation that compute functions, named transducers. In this context, I focus on class membership problems, which ask, given a program computing some function, if there is a "simpler" program which computes the same function. By "simpler", I mean that it may use less programming features, or a smaller memory, etc.
