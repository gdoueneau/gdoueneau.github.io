Research

My research deals with simple models of computation that compute functions, named transducers.
In this context, I focus on membership results. The generic question I answer is: given a machine which computes some function, is there a "simpler" machine which computes the same function ? By "simpler", I mean that it may use less programming features, or a smaller memory, etc. Apart from the theoretical point of view, this question can be useful in practice for program optimization.
